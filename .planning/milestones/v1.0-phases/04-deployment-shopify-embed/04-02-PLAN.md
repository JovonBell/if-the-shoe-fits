Create `SHOPLINE-EMBED.md` at project root with these sections:

**Section 1: Shopline Embed Snippet**
The ready-to-paste HTML snippet for Shopline's custom HTML block:

```html
<!-- If The Shoe Fits — Foot Scanner Embed -->
<div style="width: 100%; max-width: 100%; overflow: hidden;">
  <iframe
    src="https://scan.iftheshoefits.co"
    allow="camera *"
    style="width: 100%; height: 100vh; border: none;"
    title="Foot Scanner — If The Shoe Fits"
  ></iframe>
</div>
```

Key details in the snippet:
- `allow="camera *"` — required for cross-origin camera access in iframe
- `height: 100vh` — scanner needs full phone screen height
- `border: none` — seamless embed
- `title` attribute for accessibility
- No `sandbox` attribute (sandbox blocks camera by default)
- No `loading="lazy"` (scanner should load immediately when user reaches the section)

**Section 2: Vercel Deployment Steps**
Step-by-step for deploying to Vercel:
1. Go to vercel.com and sign in (or create account)
2. Click "Add New Project" -> "Import Git Repository"
3. Select the if-the-shoe-fits repository
4. Framework preset: Next.js (auto-detected)
5. Add environment variables from .env.local.example:
   - `NEXT_PUBLIC_SUPABASE_URL` — from Supabase Dashboard -> Project Settings -> API
   - `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY` — from same location (anon/public key)
   - `SUPABASE_SERVICE_ROLE_KEY` — from same location (service_role key, keep secret)
6. Click "Deploy"
7. Vercel will build and deploy automatically

**Section 3: Custom Domain Setup**
Instructions for scan.iftheshoefits.co:
1. In Vercel project -> Settings -> Domains -> Add "scan.iftheshoefits.co"
2. Vercel will show a CNAME record to add
3. Go to domain registrar (wherever iftheshoefits.co DNS is managed)
4. Add CNAME record: `scan` -> `cname.vercel-dns.com` (Vercel will show exact value)
5. Wait for DNS propagation (usually 5-30 minutes)
6. Vercel auto-provisions SSL certificate once DNS resolves

**Section 4: Shopline Theme Editor Instructions**
How to add the embed in Shopline:
1. Log in to Shopline admin panel
2. Go to Online Store -> Themes -> Customize (on active theme)
3. Navigate to the page where the scanner should appear
4. Add a new section -> "Custom HTML" (or "Custom Code" depending on Shopline theme)
5. Paste the embed snippet from Section 1
6. Save and publish
7. Test on a phone — open the page and verify camera permission prompt appears

**Section 5: Testing Checklist**
After deployment:
- [ ] Visit scan.iftheshoefits.co directly — scanner loads, camera works
- [ ] Visit Shopline store page with embed — scanner loads inside iframe
- [ ] Camera permission prompt appears inside iframe on iOS Safari
- [ ] Camera permission prompt appears inside iframe on Android Chrome
- [ ] Admin portal accessible at scan.iftheshoefits.co/admin/login
- [ ] Full scan flow works: capture -> process -> results -> lead form

**Section 6: Troubleshooting**
Common issues:
- "Camera not working in iframe" — Check that `allow="camera *"` is on the iframe tag (not `allow="camera"` without the asterisk)
- "Blank iframe" — Verify scan.iftheshoefits.co loads standalone first
- "Build failed on Vercel" — Check environment variables are set correctly
- "DNS not resolving" — CNAME propagation can take up to 48 hours; check with `dig scan.iftheshoefits.co`
